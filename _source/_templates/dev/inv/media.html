Video File

Plays a video file. To allow users to control the video, add the `controls` attribute to the `video` tag.

<div class="c_video">
  <video data-lazy-load data-src="https://starter.wbrowar.com/video/clip.mp4" controls></video>
  <noscript><video src="https://starter.wbrowar.com/video/clip.mp4" controls></video></noscript>
</div>

Iframe Video (YouTube, Vimeo, etc ...)

An iframe used to embed video content. Wraps the iframe in a `video` class to resize video in a 16:9 ratio.

<div class="c_video">
  <iframe data-lazy-load data-src="https://www.youtube.com/embed/G4Sn91t1V4g?rel=0&amp;controls=0&amp;showinfo=0" width="853" height="480"  frameborder="0" allowfullscreen></iframe>
  <noscript><iframe src="https://www.youtube.com/embed/G4Sn91t1V4g?rel=0&amp;controls=0&amp;showinfo=0" width="853" height="480"  frameborder="0" allowfullscreen></iframe></noscript>
</div>

Video Background

Fills the background of an element with a responsive video file that autoplays and loops.

<div class="c_video_bg">
  <video class="c_video_bg__video" data-lazy-load data-src="https://starter.wbrowar.com/video/clip.mp4" preload autoplay muted loop></video>
</div>
